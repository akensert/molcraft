**Transformers** and other stuff for **molecule generation** and **prediction**. 

## Installation
> [!NOTE]
> Project is under development, hence subject to breaking changes.

For GPU users:
```
git clone git@github.com:akensert/molcraft.git
pip install -e .[gpu]
```
For CPU users:
```
git clone git@github.com:akensert/molcraft.git
pip install -e .
```
